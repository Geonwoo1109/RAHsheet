# RAHsheet
1. 2025 개인 프로젝트. RAH 연습실 신청 자동화 챗봇 제작
2. google sheet API 활용

# 작동 방식
1. 공기계의 부계로 카카오톡 알림을 받는다.
2. 원하는 형태로 메시지를 가공한다.
3. 연습실 시트에서 해당 위치를 찾고 정보를 검색한다.
4. 바로 신청 가능하다면 시트에 반영하고 신청한다.
